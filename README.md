# Práctica sobre accesibilidad Web

## Autor
Andrés Iglesias Souto

## Objetivos
El objetivo de esta práctica es el de usar bootstrap para crear una página web sobre accesibilidad, siendo esta a su vez accesible y responsive

## Estructura
La página pricipal index.html está en la raiz del proyecto al igual que la secundaria

- los estilos están en una carpeta css

- las imágines están en la carpeta multimedia

- las fuentes en la carpeta fonts

- js en la raiz

## Tecnologías usadas
- html
- css
- js
- bootstrap
